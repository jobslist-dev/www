About the role

Exposure is building developer tooling that teams actually want to use — and we're looking for a Senior Software Engineer to help us build it. As an early engineer, you'll own meaningful parts of the product end to end: shaping architecture, shipping features to real developers, and setting the technical bar as we grow. This is a high-autonomy, low-bureaucracy role. You'll work across the stack, make pragmatic decisions about what to build and how, and have a direct, visible impact on the product and the team's engineering culture. We care about clean, reliable software and fast iteration based on real user feedback — not process for its own sake. If you want ownership, interesting problems, and the chance to help define how a product (and a team) is built from an early stage, let's talk.

Responsibilities

  • Design, build, and ship features across the stack — from API and data layer to UI
  • Own technical decisions in your area: architecture, trade-offs, and quality
  • Write clean, well-tested, maintainable code and review others' work
  • Partner with product and design to turn ideas into shipped, polished features
  • Improve performance, reliability, and developer experience as we scale
  • Help set engineering standards and mentor as the team grows

Requirements

  • 5+ years building and shipping production software, including a senior capacity
  • Strong full-stack experience with a modern stack (e.g., TypeScript, React/Next.js, Node)
  • Solid grasp of API design, databases, and cloud infrastructure
  • Track record of owning features end to end and making sound technical trade-offs
  • Excellent communication and a bias toward shipping and iterating
  • Comfortable with the ambiguity and autonomy of an early-stage team

Benefits

  • Fully remote with flexible hours
  • Competitive salary and meaningful early-team equity
  • High ownership and real influence on product and architecture
  • Learning, tools, and conference budget
  • Small, senior team — minimal meetings, maximum building
